Bit’s Harware

Website


I made a mock e-commerce Website with various
functionalities that were made using CSS, HTML,
JavaScript and PHP. The website consists of categories
of products that are displayed as link images on the very
front page that the user can select to purchase a specific
item. Once selected the user can log in or register as a
new customer in order to add various items into a cart.
Once added the customer can choose to either
purchase everyting in the cart or get rid of some items.

Website Code:
https://github.com/EtienneNoir/BitsHardware.git

The website includes a navigation bar and a side
menu to help users navigate the site.
JavaScript is used to enhance the website's
interactivity with animations and dynamic content display.
SQL was used to create a database that stores product images,
customer credentials, and cart items.
PHP was used to connect to the database and manipulate
the data using embedded SQL queries.
CSS was used to make the website as attractive as possible
with the sole objective of luring users to the site.

CARD SPADE

GAME


Created a functional desktop application in the form of a
card game called Spades using Java as a
programming language. The game consists of various
classes separated into their files. The Single
Responsibility Principle was used where possible to
ensure that most classes did not have overwhelming
responsibilities to improve maintainability and
readability. Each card image was also individually made
using paint, canva, and a combination of free pictures
from the internet. The code for the following game can be found on the following link:
https://github.com/EtienneNoir/SPADES.git

CALCULATOR

APP


I made a website-based calculator application and
created a tutorial for it that can be used by anyone to
put together the application and modify their own
source code. The calculator was constructed using
HTML, CSS, and JavaScript. HTML was used to define
the structure of the web application, with structures
such as div elements to organize the different sections
of the calculator, including the screen and the upper,
middle, and lower buttons. JavaScript was used to
retrieve the expression from the calculator screen,
evaluate it using the JavaScript eval function, and
display the result.
